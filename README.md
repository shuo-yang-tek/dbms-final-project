開箱即用
===

**請勿使用 IE**

[私人 Server](http://shuo-dbms-final-project.mm.my:8120)

安裝
===

*建議使用 Linux 或是 macOS 進行安裝*

* 安裝 `nodejs`

* 安裝 `npm` or `yarn`

* `yarn install` or `npm install`

更改設定
===

Server 預設監聽 `0.0.0.0:8080`

若需更改，請修改 `src/config.js`

開啟server
===

`yarn start`

or

`npm start`

===

* 楊育碩

* 熊庭緯
